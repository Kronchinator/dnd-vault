# Iron Frame
*Cursed Golem Homebrew Racial Trait*

---

## Description

When **not wearing armour**, Melogg's iron body provides natural protection.

**AC = 13 + CON modifier**

At current CON 17 (+3): **AC 16**

This replaces the standard Barbarian Unarmored Defense (10 + DEX + CON = 13), and is strictly better for Melogg.

Melogg can still use a shield with Iron Frame (+2 AC).

---

## Notes

- Melogg should **never wear armour** — Iron Frame is better than any medium or light armour available at low levels.
- AC scales with CON increases — Melogg gets better as CON goes up.
- Stacks with shield if equipped.
