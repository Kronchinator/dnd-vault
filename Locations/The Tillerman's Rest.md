# The Tillerman's Rest

## Description
The only inn in Aldenmire. Larger than it looks from the street — a wide common room, flagstone floor, heavy oak tables, a bar running the full length of the back wall. There is a fire that has never been let out. The ale is local and honest. The stew is whatever was cheap at market.

Smells like woodsmoke, tallow candles, and something always stewing in the kitchen.

## Notable Features
- **The Fire** — Always burning, low and solid, in a wide stone hearth. The warmth hits you when you open the door.
- **The Bar** — Heavy oak, provides half cover in combat. Oswin's good bottle is kept underneath, out of sight.
- **The Wobbly Table** — Corner table with a leg propped up with folded cloth. Oswin keeps meaning to fix it.
- **The Back Door** — Leads to a narrow alley and then the riverside. Useful for escaping or flanking.
- **The Kitchen** — Oswin runs it alone most nights. A door behind the bar.
- **Six Rooms Upstairs** — Simple, clean. Straw mattress, wool blanket, a candle.

## Combat Layout
```
[ KITCHEN DOOR ]     [ BACK DOOR → riverside ]
|                                              |
|   [ TABLE ]    [ TABLE ]    [ TABLE ]        |
|                                              |
|   [ TABLE ]         [ FIREPLACE ]            |
|                                              |
|==========[ BAR COUNTER ]===================|
|   (Oswin's side)                            |
[ FRONT DOOR ]
```
- Bar counter: half cover (+2 AC)
- Fireplace side: dim light, concealment
- Tables: can be flipped as bonus action for half cover
- Back door: escape or flanking route
